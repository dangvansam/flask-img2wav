# flask-img2wav
**Image - Audio(Spectrogram) Translation, Flask Server, PHGI Phase Recovery From Spectrogram**  
**Note**  
Install **ltfatpy** (only support Linux, macOs):  
sudo apt install libfftw3-dev liblapack-dev  
pip install ltfatpy  
**Run server:**  
python flask_img2wav.py
